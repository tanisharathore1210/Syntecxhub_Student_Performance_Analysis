# Student Performance Analysis Dashboard

## 📌 Project Overview
This project was completed as part of my Data Analytics Internship at Syntecxhub. The dashboard was built using Microsoft Power BI to analyze student performance and generate meaningful insights.

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel (CSV Dataset)

## 📊 Dashboard Features
- Student Count KPI
- Average Math Score
- Average Reading Score
- Average Writing Score
- Overall Average Score
- Performance Analysis by Gender
- Performance Analysis by Ethnicity
- Performance by Parental Education
- Test Preparation Analysis
- Top 5 Students Performance
- Interactive Slicers and Filters

## 📈 Key Insights
- Compared student performance across different demographic groups.
- Analyzed the impact of parental education on academic performance.
- Evaluated the effect of test preparation on student scores.
- Identified top-performing students using interactive visualizations.
